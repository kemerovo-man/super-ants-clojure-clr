Installation Super Ants
------------------------
1. Download files (for example to C:\super-ants)
2. Download latest version of ClojureCLR http://sourceforge.net/projects/clojureclr/files/ (for example to C:\clojure-clr)
3. Add or modify environment variable CLOJURE_LOAD_PATH to C:\super-ants
4. Run Clojure.Main.exe
5. Type in ClojureCLR console: (load "start")
Or create shortcut with follow property "C:\clojure-clr\Clojure.Main.exe" "C:\super-ants\start.clj"
Modify paths.
6. Enjoy!
