This is a ClojureCLR ants demo evolution by Evgeny Antipov (kemerovo_man@mail.ru).
Based on https://github.com/clojure/clojure-clr/blob/master/Clojure/Clojure.Source/clojure/samples/ants.clj
by David Miller.

Installation
------------------------
1. Download super-ants-clojure-clr files
1. Download latest version of ClojureCLR http://sourceforge.net/projects/clojureclr/files/
2. Run Clojure.Main.exe
3. Type in ClojureCLR console:
 (load-file "path-to-super-ants/main.clj")
or create shortcut with follow property "path-to-clojureCLR\Clojure.Main.exe" "path-to-super-ants/main.clj"
modify paths.
4. Enjoy!
